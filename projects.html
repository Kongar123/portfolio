Featured Projects

Here are some of the projects I'm proud to have worked on.

Hotel Management System (Java)

A comprehensive hotel management system built using Java and OOP principles, designed to streamline hotel operations. The system manages room reservations, check-ins, check-outs, customer records, room availability, and billing. It features a clean Swing-based GUI, efficient data handling, and a modular architecture that makes it easy to maintain and extend. This project demonstrates strong skills in Java, file handling, and user-friendly interface design.

GPA Calculator (Java)

A desktop GPA Calculator application developed in Java using Swing, created to help students calculate their academic GPA with ease and accuracy. The system allows users to add courses, assign credit hours, select grades, and instantly compute the overall GPA. It includes input validation, dynamic table updates, exporting the project as an executable JAR, and a polished interface. This project highlights expertise in Java UI development, event handling, and building real-world academic tools.
